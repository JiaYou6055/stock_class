# 台股產業分類與題材追蹤系統 (Stock Class Tracker)

這是一個自動化的台股資料抓取與分類系統，旨在整合證交所（TWSE）的官方產業分類與玩股網（WantGoo）的市場題材、概念股分類，建立一個更精確的投資研究資料庫。

## 🌟 核心功能

- **多來源資料整合**：結合證交所官方產業分類與民間投資網站的熱門概念題材。
- **自動化爬蟲**：定期從 TWSE 與 玩股網 抓取最新的股票名單與題材分類。
- **精細化產業標籤**：除了基礎產業，特別強化了半導體供應鏈、電子零組件及特定熱門產業（如航太、軍工、ABF載板等）的細分標籤。
- **資料同步機制**：內建檢查與更新邏輯，確保本地 JSON 資料庫與市場同步。

## 📂 專案結構

```text
├── src/
│   ├── data/               # 核心資料庫 (JSON 格式)
│   │   ├── all_stocks.json      # 全市場股票基本資料
│   │   ├── stock_index.json     # 股票代碼索引
│   │   ├── themes.json          # 精細化題材分類 (重點維護對象)
│   │   └── twse_industries.json # 證交所官方產業分類
│   ├── scrapers/           # 爬蟲程式
│   │   ├── twseScraper.js       # 證交所官方資料抓取
│   │   └── wantgooScraper.js    # 玩股網熱門題材抓取
│   └── utils/              # 工具程式
│       ├── checkAndUpdateData.js # 檢查資料完整性
│       └── syncData.js          # 資料同步主程式
├── package.json
└── README.md
```

## 🏷️ 精選細分標籤 (Thematic Tags)

本系統特別優化了以下專業領域的分類，超越傳統的寬泛產業分類：

- **半導體系列**：IC設計、晶圓代工、半導體封測、半導體檢測、半導體設備、半導體耗材、半導體廠務、功率半導體。
- **電子零組件**：ABF載板、PCB (印刷電路板)、被動元件、銅箔基板 (CCL)。
- **科技前瞻**：AI伺服器、III-V族 (砷化鎵)、品牌ODM、網通系統廠。
- **傳產轉型與國防**：航太軍工、車用零組件、自行車。

## 🚀 快速上手

### 安裝環境
確保您的系統已安裝 [Node.js](https://nodejs.org/)。

```bash
# 安裝依賴套件
npm install
```

### 1. 資料更新 (爬蟲)
執行以下指令來抓取證交所與玩股網的最新分類資料：

```bash
node src/utils/syncData.js
```

### 2. 啟動互動式查詢系統
執行主程式進入查詢模式：

```bash
node src/index.js
```

## 📖 使用範例 (Examples)

進入系統後，您可以透過以下方式進行查詢：

### 範例 A：查詢特定股票代號
直接輸入代號即可查看其市場分類與我們補強的專業題材。
```text
查詢> 3037

[3037] 欣興電子股份有限公司
市場: 上市
產業: 電子零組件業
相關題材: 電子零組件, 電子零組件業, ABF載板, PCB, 印刷電路板
```

### 範例 B：查詢特定題材成份股 (關鍵功能)
使用 `theme [名稱]` 指令，找出該產業鏈的所有相關股票。
```text
查詢> theme ABF載板

題材 [ABF載板] 的成份股 (4 檔):
3037 - 欣興電子股份有限公司
3189 - 景碩科技股份有限公司
4958 - 臻鼎-KY
8046 - 南亞電路板股份有限公司
```

### 範例 C：關鍵字模糊搜尋
如果您不記得代號，可以直接輸入名稱。
```text
查詢> 漢翔

找到 1 筆相似結果:
2634 - 漢翔航空工業股份有限公司 (航運業)
```

### 範例 D：查詢系統中所有包含「半導體」的細分題材
如果您不確定精確的題材名稱，輸入 `theme` 關鍵字，系統會提供建議。
```text
查詢> theme 半導體

找不到精確題材 "半導體"，你是指？
- 半導體
- 半導體業
- 半導體檢測
- 半導體設備
- 半導體耗材
- 半導體廠務
- 功率半導體
```
